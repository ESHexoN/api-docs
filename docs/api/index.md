# API 接口示例

> 多平台情况下，WeatherAllInOne 仅返回同一格式的数据，方便调用。

## 今日天气 (`/api/today`)

> 返回今日天气（当前天气）。


```json
{
    "code": 200, // 状态码
    "data": {
        "weather": "小雨", // 当前天气情况
        "temp": {
            "now": "19", // 现在温度
            "day": "22", // 日间温度（最高温度）
            "night": "16" // 夜间温度（最低温度）
        },
        "wind": [
            "西南风", // 风向
            "1级" // 风力
        ],
        "aq": [
            "16", // 空气指数
            "优" // 空气值
        ],
        "humidity": "100%", // 相对湿度
        "sun": {
            "sunrise": "06:35", // 日出
            "sunset": "17:46" // 日落
        }
    }
}
```
